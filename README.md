# Rasteriser and Ray Tracer

## Ray Tracer
### Features
* Extensible design
* Soft shadows
* Depth of field
* Multiple light sources
* Anti-aliasing
* Phong lighting
* Back-face culling
* Object importer
* Global illumination (Monte-Carlo)
* Mirrors
* Glass (Refraction)

### Optimisations
* Moller-Trumbore intersection
* K-D Tree
